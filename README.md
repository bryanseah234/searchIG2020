# Instagram Profile Search Tool

A Python script to search and scrape Instagram user profiles based on keywords found in their bio.

## Description

This project leverages Google search to find Instagram profiles that contain specific keywords in their bio. It automates the process of discovering Instagram users with particular interests or characteristics by using targeted Google search queries and saves the found profile URLs to a text file.

## Features

- Search for Instagram profiles using custom keywords
- Leverages Google search for discovering profiles
- Automatic rate limiting to avoid blocking
- Saves discovered profile URLs to a text file for later use

## Technologies Used

- Python
- Google Search library (`googlesearch-python`)

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/searchIG2020.git

# Navigate to project directory
cd searchIG2020

# Install dependencies
pip install googlesearch-python
```

## Usage

```bash
# Run the script
python "scrape instagram code.py"
```

**Steps:**
1. Open `scrape instagram code.py` in an editor
2. Replace `X` in the queries variable with your desired keyword(s)
3. Save the file
4. Run the script using the command above
5. Found Instagram profile URLs will be saved to `urls.txt`
